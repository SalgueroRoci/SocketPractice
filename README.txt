Python 2.7

Place the server program on a different directory than the client directory to test. 
Start the server program first by opening the terminal on linux 
Type "python ftpserver.py" to start the server 

Then start the client open a new terminal and "python ftpcli.py <server name> <port number>"

Server connection tested on localhost (if client and server on same machine) and port = 5021
After connection is established 'ftp>' will show up and 
the commands ls, put <filename>, get <filename>, quit will be available
